## Talos
Talos is composed of:
* a Mother Board
  * a 32-bit CPU
    * single Core
        * ALU
        * 16 registers
  * a from 1 to 516Kb RAM
  * a ROM where all the instructions (32-bit) are stored
* a ASM interpreter to convert assembly code to 32-bit bytecode
